Struct fastcrypto::groups::bls12381::Scalar

source · 
pub struct Scalar(/* private fields */);
Expand description

This represents a scalar modulo r = 0x73eda753299d7d483339d80809a1d80553bda402fffe5bfeffffffff00000001 which is the order of the groups G1, G2 and GT. Note that r is a 255 bit prime.
