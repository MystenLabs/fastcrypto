WindowedScalarMultiplier
This scalar multiplier uses pre-computation with the windowed method. This multiplier is particularly fast for double multiplications, where a sliding window method is used, but this implies that the double_mul, is NOT constant time. However, the single multiplication method mul is constant time if the group operations for G are constant time.

multi_scalar_mul
This method computes the linear combination of the given scalars and group elements using the sliding window method. Some group elements may have tables of precomputed elements which can be given in the precomputed hash map. For the elements which does not have a precomputed table a table of size 2default_window_width - 1 is computed.
