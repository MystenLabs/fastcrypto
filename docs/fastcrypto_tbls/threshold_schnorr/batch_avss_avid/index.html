Module batch_avss_avid 

Source
Expand description

Asynchronous verifiable secret sharing (AVSS) for a batch of random nonces.

A dealer shares L = w_dealer · BATCH_SIZE nonces among n weighted receivers with total weight W under a threshold t. The numbered steps below, starting at Dealer::create_avss_messages, walk through the protocol.
