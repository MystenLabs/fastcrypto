Verify Groth16 proof using the serialized form of the four components in a prepared verifying key (see more at PreparedVerifyingKey), serialized proof public input, which should be concatenated serialized field elements of the scalar field of [crate::conversions::SCALAR_SIZE] bytes each in little-endian format, and serialized proof points.
