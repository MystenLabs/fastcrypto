Structs

  • StrongFiatShamir
    Implementation of the Fiat-Shamir challenge generation for usage with Wesolowski’s VDF construction. The implementation is strong, meaning that all public parameters are used in the challenge generation. See https://eprint.iacr.org/2023/691.
  • WeakFiatShamir
    Implementation of the Fiat-Shamir challenge generation compatible with chiavdf. Note that this implementation is weak, meaning that not all public parameters are used in the challenge generation. This is not secure if an adversary can influence the public parameters. See https://eprint.iacr.org/2023/691.
  • WesolowskiVDF
    An implementation of the Wesolowski VDF construction (https://eprint.iacr.org/2018/623) over a group of unknown order.
